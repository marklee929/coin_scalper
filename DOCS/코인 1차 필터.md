# 반복 박동 트레이딩 – 패턴 1단 (Binance Spot 기준)

## 0. 전제 고정 (절대 변경 금지)

* **목표**: 꾸준한 수입
* **대상**: 중형 알트 (Binance Spot 상장 코인)
* **방식**: 반복 박동
* **배제**

  * 한방 트레이드
  * 신규 상장 첫날
  * 서사·뉴스·이슈 추격

👉 모든 판단은 이 전제 안에서만 한다.

---

## 1. 마켓 선택 규칙

* **BINANCE SPOT만 사용**
* Alpha / 선물 / 파생 전부 배제

**이유**

* Spot = 사람 + 봇 + 단타가 섞인 “생활 시장”
* 패턴이 반복되고 데이터가 누적됨
* 이벤트성 왜곡이 상대적으로 적음

👉 패턴 제작은 Spot이 정석
👉 다른 마켓은 나중에 “검증용”으로만 고려

---

## 2. 마켓 진입 후 첫 행동 (절대 금지 vs 필수 행동)

### ❌ 하지 말 것

* 상승률 Top
* 거래량 급증
* 오늘 핫한 코인
* 랭킹 / 추천 / 메인 화면

👉 전부 **결과 화면**임

---

### ✅ 해야 할 것

* **24h Change % 하단 스캔**
* Binance Spot 전체 심볼
* **알파벳 순서로 강제 스캔**

#### 가격 구간 기준

```
-5% ~ -20%  → 관찰 대상
```

#### 즉시 제외

* -1% ~ -3% : 아직 안 털림
* -40% 이하 : 구조 붕괴 가능성

**이 구간의 의미**

* 이미 한 번 털림
* 추격 수요 제거됨
* 다시 움직일 수 있는 후보만 남음

---

## 3. 하락 리스트 1차 컷 (즉시 제거 조건)

아래 중 **하나라도 해당되면 바로 제거**

* 최근 **상장 1~2일**
* 차트에 **장대봉 1~2개만 존재**
* 거래량이 사실상 0
* 고점 대비 **-70~-90%**인데

  * 중간 반등 흔적 없음

👉 이건 **죽은 코인**
👉 관찰 가치 없음

---

## 4. 2차 행동 – 차트 프레임 고정

### 프레임

* **1H / 4H**
* ❌ 분봉 절대 금지

### 여기서 찾는 구조

```
고점 → 하락 → 횡보
```

* 횡보 기간: **2 ~ 5일**
* 바닥 구간에서 확인할 것

  * 거래량이 완전 0은 아님
  * 고저점이 아주 조금씩 정리됨

---

## 5. 이 단계에서의 질문 규칙

### ❌ 하지 않는 질문

* 오를 것 같나?
* 지금이 바닥 아닌가?
* 내일 오르나?
* 이거 서사 있나?

### ⭕ 유일한 질문

```
사람이 아직 남아 있나?
```

* 거래가 이어지고 있는가
* 완전히 버려진 차트는 아닌가
* “살아있는 횡보”인가

---

## 6. (선택) Alpha 확인 타이밍 – 후순위

> **필수 아님 / 옵션**

### 조건

* Spot 기준 후보가 **2~3개 남았을 때만**
* 전체 스캔 중에는 절대 안 봄

### 확인 항목

* Alpha에도 같은 코인이 있는가
* Alpha에서 이미 한 번 털렸는가
* 상장 프리미엄이 끝났는가

### 판단

* Spot 구조 ≈ Alpha 구조 → 유지
* Alpha만 과열 → 제외

---

## 7. 패턴 1단의 성격 (중요)

* **이 단계는 매수 단계가 아니다**
* 관찰 리스트를 만드는 단계다

### 기록만 남긴다

* 코인 심볼
* 횡보 구간
* 지지 가격대
* 거래량 상태

---

## 8. 패턴 1단 종료 조건

* 후보 **2~3개만 남김**
* **매수 없음**
* 차트 + 가격대 + 시간만 기록
