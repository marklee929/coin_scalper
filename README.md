# coin_scalper (Binance Spot)

Binance Spot 기반 **눌림(하락→횡보) 구조**를 스캔하고,  
관찰 리스트(watchlist)를 만든 뒤 **보수적으로 진입 → 일정 수익/구조 붕괴 시 즉시 종료**하는 자동매매 실험 프로젝트.

> 목표: **대박이 아니라 꾸준한 현금흐름**  
> 방식: **예측하지 않고, 반복 가능한 행동만 수행**  
> 시장: **Binance Spot only (Alpha/Futures 사용 안 함)**


## Strategy (Pattern Stage 1)
- 대상: Binance Spot 중형 알트(USDT 마켓 중심)
- 스캔 방식: **전체 심볼 알파벳 순 강제 스캔** (FOMO/뉴스 차단)
- 1차 필터(즉시 제거):
  - 신규 상장 1~2일
  - 거래량 사망
  - ATH 대비 -70~-90% + 반등 흔적 없음
  - 장대봉 몇 개만 있는 차트
- 가격 구간 필터(24h):
  - **-5% ~ -20%만 관찰**
  - -1~-3% 제외(아직 안 털림)
  - -40% 이하 제외(붕괴 위험)
- 차트 프레임 고정: **1H / 4H** (분봉 금지)
- 찾는 구조: **고점 → 하락 → 횡보(2~5일)**  
- 질문: “오를까?” ❌ / “사람이 남아있나?” ⭕

> Stage 1 종료 조건: 후보 2~3개만 남기고 **매수하지 않음**(기록만)


## Risk Rules
- Spot 거래만 사용 (Withdraw/Margin/Futures 권한 비활성 권장)
- 자본 운용은 **1/N 분할** (N = 동시 보유 포지션 수)
- 일정 수익 도달 또는 구조 붕괴 시 **즉시 종료**
- 종료 후 재진입은 제한(설정 기반)
